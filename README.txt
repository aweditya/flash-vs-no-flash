Dataset Folder contains code files and input images for Flash and Ambient Illuminations Dataset image denoising and detail transfer
Microsoft Folder contains code files to run on images from the original Microsoft Research paper
cameraToXYZtoSRGB.m file is provided by the FAID to perform color mapping under CC-BY-NC-SA

The Microsoft Research paper images being .tif files occupy large amount of space (3 flash/no-flash pairs are almost 70MB)
